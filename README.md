# static-html-template

The workflow is Use this template to create static HTML websites and deploy them on Render.

****Workflow: Codespaces → GitHub → Render**
This presents a custom personal Website created using HTML and CSS, and is hosted on Render.
The workflow that was used started with Github Codespaces and was then deployed onto the Render Server.
The basic structure includes a standard header and body sections. 
The styling rules are within the body section. The code was validated using the HTML validator given in the lesson.
